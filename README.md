## Explainable AI for Diabetes Prediction

This project predicts diabetes risk using machine learning and provides model explanations using SHAP, LIME, and counterfactual explanations.

## Model
XGBoost Classifier

## Explainability Methods
- SHAP Beeswarm Plot
- SHAP Waterfall Plot
- LIME Local Explanation
- Counterfactual Explanation

## Dataset
Pima Indians Diabetes Dataset

## Run the App

streamlit run diab.py

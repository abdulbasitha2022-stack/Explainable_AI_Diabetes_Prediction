# Explainable AI for Diabetes Prediction

🚀 An end-to-end explainable AI system that predicts diabetes risk and provides interpretable, actionable insights using SHAP, LIME, and interactive simulation.

---

## Overview
This project builds a diabetes prediction system using XGBoost and integrates explainable AI techniques to improve transparency and usability in healthcare.

The system goes beyond prediction by including:
- Model explainability (SHAP, LIME)
- What-if simulation for feature sensitivity
- Personalized health recommendations
- Counterfactual explanations for actionable insights

---

## Key Features
- XGBoost-based diabetes prediction model  
- SHAP for global and local feature importance  
- LIME for interpretable local explanations  
- Counterfactual analysis for actionable feature changes  
- What-if simulation to analyze feature impact on predictions  
- Rule-based health recommendation system  
- Interactive Streamlit application  

---

## Results
- Identified glucose and BMI as dominant risk factors  
- Enabled patient-level interpretability using SHAP and LIME  
- Provided actionable recommendations based on input features  
- Demonstrated model sensitivity through interactive simulation  

---

## Demo Features
- Real-time diabetes risk prediction  
- SHAP waterfall and summary plots  
- What-if simulation graphs  
- Personalized health recommendations  

---

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, SHAP, LIME, Streamlit, Matplotlib, Seaborn
